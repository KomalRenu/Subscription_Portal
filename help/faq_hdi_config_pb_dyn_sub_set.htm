Configure page-by questions for dynamic subscription sets

Steps

  1. In the Services Configuration section, on the Welcome page, review the Default Service Configuration settings.

  2. Click Next.
    Note: If you have already configured one or more services, proceed with the following steps. If you have not configured any services, skip step 3 and proceed directly to step 4.
  1. On the Services page, click the Configure another service link.

  2. Navigate through the folders until you locate the application object that represents the service to be configured. Select the object and click Next to proceed to the Dynamic Subscriptions page.

  3. Click the configure link next to the the dynamic subscription set to be configured.

  4. Select the answer to be used for each page-by question.
    Note: For each question for which you choose "answer to another question," you need to select an alternative question by following the steps described in Select an alternate question.
  1. Configure any additional questions you want the user to answer when creating a subscription (optional).

  2. Proceed with mapping the subscription information to the data warehouse tables.
