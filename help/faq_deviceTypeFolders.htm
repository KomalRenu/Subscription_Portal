This page allows you to select the different folders of devices associated with the device type you selected.

You navigate through the various folders and click the folders you want to be selected for this device type. Move the folders to the right of the table by using the toggle buttons. Click Next when you are done.

Narrowcast Server provides you with many predefined device types from which you can select. You can also create your own device types. To move up in the hierarchy and select different folders, click the folder name next to the You are here: link and continue to navigate until all the folders you want to add have been selected.
