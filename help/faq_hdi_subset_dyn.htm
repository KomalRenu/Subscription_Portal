Configure a Static Subscription

Steps

  1. Select the means for answering page-by questions.
  2. Click Configure Services.
  3. Click Next.
  4. Navigate through the folders until you locate the application object that represents the service to be configured. Select the object and click Next.
  5. Select an existing subscription set.
  6. Click Next.
  7. Set the options for the page-by questions.
  8. Configure any additional questions for subscribers.
  9. Click Next.
  10. Set storage locations. Click Next.
  11. Set tables for question mappings. Click Next.
  12. Map question answers to database tables. Click Next.
  13. Repeat steps 6 through 12 for all question objects or subscription sets.
  14. Having configured all subscriptions sets, click Next to return to the Configure Static Subscription Set page.

Note: Click Exit Configure Services to exit the process. Any changes already made are saved.

Map Question Answers to Database Tables

Steps

  1. Select the data warehouse from the drop-down list.
  2. Click Refresh to refresh the list of available tables associatedwith the selected data warehouse.
  3. Highlight the table(s) to be present in the warehouse from the Available Tables list.
  4. Using the toggle switches, move the tables to the left and right until all desired tables are present in the Selected Tables box.

Move Answer Components to Database Tables

Steps

  1. Click the answer component to be placed in the table.
  2. Drag the component to the desired table and release it in the specified column.
