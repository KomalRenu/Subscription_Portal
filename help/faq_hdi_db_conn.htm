Add a database connection

The first time you create a portal, the list of database connections for all databases (Portal Repository, Object Repository and Subscription Book Repository) is blank. You need to add a new one.

Steps

  1. In the System Configuration section, on the Add a database connection page, type a name for the Database Connection.
  2. Select the type of database from the drop-down list.
  3. Type the name of the computer on which the SequeLink Server is running in the SequeLink server name box.
  4. Enter the ODBC DSN as defined on the JDCS computer in the ODBC Data Source Name box.
  5. Enter the Login ID that the Subscription Portal should use to connect to this database.
  6. Enter the Password that corresponds to this User ID.
  7. Enter the password again in Confirm password.
  8. Click Add Database Connection to add the database connection.
Note: The name for the database connection can be any name you want.
