Create a new device type

Steps

  1. In the Site Preferences section, on the Device Types page, click Create new device type.
  2. Type the new device type name.
  3. Type the URLs of the large and small icons.
  4. Select the address format.
  5. Select the fields to use when displaying and editing.
  6. Click Next.
