Edit a database connection

From the Portal Repository, Object Repository or Subscription Repository page, you can edit the database connection.

Steps

  1. In the System Configuration section, click Edit next to the database connection to be modified.
  2. Modify the information as necessary.
  3. Enter the Password that corresponds to this User ID.
  4. Enter the password again in Confirm password.
  5. Click Update Database Connection to add the database connection.
Note: The name for the database connection can be any name you want.
