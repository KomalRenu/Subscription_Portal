Add a new site definition

Steps

  1. In the Site Management section, on the Site Definition page, click Add a new site definition.
  2. Type a Name for the site definition.
  3. Type a Description of the site.
  4. Click Next.
  5. Select the database connection where the Object Repository is stored.
  6. Enter the table prefix for the Object Repository.
  7. Select the Use same database connection for the Subscription Book Repository check box. Click Next.
Note:

  • If the Subscription Book Repository is in a different location, clear the check box and click Next. On the Subscription Book Respository page, enter the database connection where the Subscription Book Repository is stored and the table prefix for the SBR.
