Rename Device Types

Steps

  1. Click rename... next to the device type you wish to rename.
  2. Enter in the new name of the device.
  3. Click Rename.

Reconfigure Device Types Definitions

Steps

  1. Click definition... next to the device type you wish to reconfigure.
  2. Enter in the URLs of the large and small icons.
  3. Select the address format.
  4. Select the fields to use when displaying and editing.
  5. Click Next.

Delete Device Types

Steps

  1. Click Delete next to the device type you wish to delete.
  2. Click Yes to confirm that you wish to delete it.

Edit the Device Folder

Steps

  1. Click Edit device folder next to the device type whose folder you wish to edit.
  2. Select an available item on the left that you wish to make active for this folder.
    3. >Click Next.

Create New Devices

Steps

  1. Click Create New Device Type.
  2. Enter the new device type name.
  3. Enter the URLs of the large and small icons.
  4. Select the address format.
  5. Select the fields to use when displaying and editing.
  6. Click Next.
