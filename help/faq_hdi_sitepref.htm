Add a New Channel

Steps

  1. Click Add a New Channel.
  2. Select the Channels Folder to be displayed in the Subscription Portal.
  3. Click Next.
  4. Enter a name for this channel.
  5. Enter a description of channel.
  6. Select the Channel Enabled option (optional).

Modify the Name or Description of the Channel Folder

Steps

  1. Select the folder to be modified.
  2. Click Next.
  3. Enter a name for this channel.
  4. Enter a description of channel.
  5. Select the Channel Enabled option (optional).
  6. Click Next.

Delete a Channel

Deleting a channel is a non-reversable process. If you plan to reuse the channel or one similiar to it, it is better to modify the channel or keep it instead of deleting it.

Steps

  1. Click the Delete link next to the channel to be deleted.
  2. At the confirmation window, click Yes to delete the channel, or click No to cancel the process and return to the Channel page.
