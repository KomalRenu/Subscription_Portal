Create a Database Connection

The first time you create a portal, the list of database connections for all databases (Portal Repository, Object Repository and Subscription Book Repository) is blank. You need to add a new one.

Steps

  1. Select the type of database from the list.
  2. Enter the name of the Database Connection.
  3. Enter the SequeLink server name in the SequeLink Server box.
  4. Enter the ODBC DSN name in the ODBC Data Source Name box.
  5. Enter the Login ID to log on the database.
  6. Enter the Password to log on the database.
  7. Enter the password again in the Confirm password box.
  8. Enter the desired number of pooled connections in the Number of pooled connections box.
  9. Click Add DB Connection to add the database connection to the database.

Select a Portal Repository

Steps

  1. Select the database connection that contains the Portal Repository.
  2. Enter the Tables prefix found in the Portal Repository.
  3. Click Next.
