Engine Configuration Interface

What is it?

The Engine Configuration page allows you to configure the Portal Administrator for the creation of subscription portals. Enter the location of the Subscription Engine server, and select a database connection for the Portal Repository.

How do I access it?

Open the Portal Administrator Wizard and click the Engine Configuration link at the top of the page. The Welcome page appears. Click Next to proceed.

What can I do with it?

The Engine Configuration page lets you

  • specify the location of the Subscription Engine
  • choose a database connection for the Portal Repository
  • create a new database connection

For instructions on how to perform this task, see the following How Do I... ? topics:

  • Portal Repository
  • Database Connections

What should I know before I use it?

You should understand how the Subscription Engine was installed, the purpose of the Portal Repository, and how to add a database connection.
