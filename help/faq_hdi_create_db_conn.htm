Create a database connection

The first time you create a portal, the list of database connections for all databases (Portal Repository, Object Repository and Subscription Book Repository) is blank. You need to add a new one.

Steps

  1. Select the type of database from the list.
  2. Type a name for the Database Connection.
Note: The name for the database connection can be any name you wish to call it.
  1. Type in the name of the computer on which the SequeLink Server is running in the SequeLink server name box.
  2. Enter the ODBC DSN as defined on the JDCS computer in the ODBC Data Source Name box.
  3. Enter the Login ID that the Subscription Portal should use to connect to this database.
  4. Enter the Password that corresponds to this User ID.
  5. Enter the password again in the Confirm password box.
  6. If you wish to test the database connection, select the Test database connection check box.
  7. Click Add DB Connection to add the database connection to the database.
