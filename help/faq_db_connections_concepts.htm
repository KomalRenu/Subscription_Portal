Database connections

The first time you configure Subscription Portal, you must create a database connection for each of the databases to which you want to connect. To do this, click Add a new database connection on any of the three repository pages. Subscription Portal connects to three information repositories:

  • Portal Repository
  • Object Repository
  • Subscription Book Repository

These information repositories, which can reside in the same database or separate databases, are described in detail in their corresponding sections.

When you create a new database connection, the information entered is the same for all three repositories:

  • a name by which you want to refer to the connection you are creating
  • the type of the database (SQL Server, Oracle, DB2, or Teradata)
  • the name of the machine on which JSequeLink Server is installed and running
  • the name of the ODBC DSN on the SequeLink Server machine which corresponds to the database to which you want to connect
  • the login ID and password that the Subscription Portal should use when it accesses that database

Once you click Add a database connection you are returned to the appropriate Repository page and you can select the connection you just created.
