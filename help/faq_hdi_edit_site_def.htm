Edit a site definition

Steps

  1. In the Site Management section, on the Site Definition page, select a site from the list.
  2. Click Edit.
  3. Edit the Name and Description boxes.
  4. Click Next.
  5. Select the Object Repository.
  6. Click Next.
  7. Select the Subscription Book Repository.
