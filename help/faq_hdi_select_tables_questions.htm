Select tables for question mappings and subscription mappings

Steps

  1. In the Services Configuration section, on the Select Tables page, select the database tables from the list of available database tables.
  2. Click Refresh to refresh the list of available tables associated with the selected data warehouse.
  3. Highlight the tables to be used from the Available Tables list.
  4. Using the toggle switches, move the tables to the left and right until all the desired tables are present in the Selected Tables box.
Note: You can use the Filter table names box to restrict the list of tables names that is returned. You must use the percent (%) sign after the letter, for example, a%, to return all entries beginning with that letter or series of letters.
