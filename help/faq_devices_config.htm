Two options exists for Device Settings:

  • Portal Device
  • Default Device

When you select the default device you must select validation for the address format. Validation occurs once the subscriber enters his information in the Subscription Portal. Three options exist for address format:

  • E-mail selects the standard electronic mail format for the device, for example, yourname@company.extension
  • Numeric allows the end user to enter a numeric address for the device type, such as that supported by a pager
  • No validation allows the end user to enter any combination of text and numbers into the address field
