Map dynamic subscription information to data warehouse tables

Steps

  1. In the Services Configuration section, on the Select Columns page, select one or more general components and drag them each to one of the table columns on the right.

    Notes:
    • The general components include all of the standard subscription fields, including Subscription_ID, User_ID and Preference_ID.
    • Subscription_ID must be included in every table.
    • Every time a subscription is created, Subscription Portal creates a new Subscription ID, which is unique to the subscription.
  1. Select one or more question components and drag them each to one of the table columns on the right.

    Note: A question component is the answer ID for each question in the service.
  1. If desired, create a custom value by typing the name of the value in the custom value box. Select the desired custom value component and drag it to the desired table column and release.

    Note: Custom values allow you to write a constant value into warehouse tables each time data is inserted.
