A channel is a grouping of services that are made available through the Subscription Portal. The end user navigates through the channels to select services to which he wishes to subscribe. Each channel is mapped to one folder of services in the Object Repository.

For more information on services, refer to the Service Wizard in the MicroStrategy Narrowcast Server Application Designer Guide.
