Configure with the Portal Administrator Wizard

  1. Use the Services Configuration page interface of the Portal Administrator Wizard to select the service that was created in Narrowcast Administrator.
  2. On the Static Subscriptions page, click Configure to define the configuration for all the static subscription sets in this service.
  3. On the Configure Subscription Set page, mark the page-by question (1.2 Page-by Information Object) as "Page-by: Subscription ID".
  4. Browse for an additional question and choose the question object 1.2 Alternate Question Object.
  5. For storage of the alternate question, choose to create a new storage mapping.
  6. Select the database connection that corresponds to your warehouse and select the SUB_THRESHOLD table created for this example.
  7. Using the available answer components, map User ID to the MR_USER_ID column of the SUB_THRESHOLD table, Subscription ID to the MR_SUB_ID column, and the Prompt Answer to the THRESHOLD column.
  8. Skip the Dynamic Subscription Configuration page for this service.
