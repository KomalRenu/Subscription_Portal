Services configuration

The diagram below illustrates the steps necessary to configure services with page-by questions.

Before you begin configuring services with page-by questions it is important that you have a strong understanding of personalized page execution (page-by) question objects and dynamic subscription sets. You should also review Advanced Subscription Portal Configuration for examples on advanced configurations and when you would use the Services Configuration section of the Portal Administrator Wizard.

It is not necessary to configure all services through the Services Configuration section of the Portal Administrator Wizard. Only those services using either personalized page execution or dynamic subscription sets need to be configured using this interface.
