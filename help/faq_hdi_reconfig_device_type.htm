Reconfigure a device type definition

Steps

  1. In the Site Preferences section, on the Device Types page, click definition next to the device type to be reconfigured.
  2. Type the URLs of the large and small icons.
  3. Select the address format.
  4. Select the fields to use when displaying and editing. Selecting any of the check boxes under Edit fields causes these fields to be displayed to the end user when he edits an address of this particular device type.
  5. Click Next to proceed to the Device Folders page.
