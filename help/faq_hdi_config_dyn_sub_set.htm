Configure a dynamic subscription set

Steps

  1. Review the Default Service Configuration settings.
  2. Click Next.
Note: If you have already configured one or more services, proceed with the following steps. If you have not configured any services, skip step 3 and proceed directly to step 4.
  1. Click the Configure another service link.
  2. Navigate through the folders until you locate the application object that represents the service to be configured. Select the object and click Next to proceed to the Static Subscriptions page.
  3. Click Next to proceed to the Dynamic Subscriptions page.
  4. Select a subscription set and click configure.
Notes:
  • You can choose to either configure dynamic subscription sets individually or define a single configuration to blanket all dynamic subscription sets.
  • If you wish to configure dynamic subscription sets individually, click the configure link next to the subscription set you wish to configure.
  • To create a single configuration, select the configure link available under the Configure dynamic subscription sets default option.
  • If you configure the default dynamic subscription set configuration, you can configure individual dynamic subscription sets as well. The individual configurations overwrite the default settings for the specific subscription set it is being configured for.
  1. Set the options for the page-by questions.
  2. Click Next.
  3. Select the warehouse and tables in which you want to store the subscriptions for the selected subscription set. Click Next.
  4. Map the subscription information to the database tables. Click Next.
  5. Repeat steps 6 through 10 for all dynamic subscription sets you wish to configure.
  6. Having configured all dynamic subscriptions sets, click Next to return to the Configured Services page.
