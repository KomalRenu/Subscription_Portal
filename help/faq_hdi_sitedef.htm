Add a New Site Definition

Steps

  1. Click Add a new site definition.
  2. Enter the Name of the site definition.
  3. Enter a Description of the site properties.
  4. Click Next.
  5. Select the Object Repository.
  6. Click Next.
  7. Select the Subscription Book Repository.
Note: For more information refer to Add a new database connection

Edit a Site Definition

Steps

  1. Select a site from the list.
  2. Click Edit.
  3. Edit the Name and Description boxes.
  4. Select the Object Repository.
  5. Click Next.
  6. Select the Subscription Book Repository.

Delete a Site Definition

Steps

  1. Select a site from the list.
  2. Click Delete.
  3. At the confirmation window click Yes.
